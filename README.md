# SmartChat AI

SmartChat AI is an AI-powered conversational chatbot that combines Machine Learning and Large Language Models (LLMs) to provide intelligent, real-time responses. The system supports text and voice interactions, multilingual communication, and intent classification using a custom-trained Machine Learning model.

## Features

* AI-powered conversational chatbot
* Intent Classification using TF-IDF and Logistic Regression
* Voice Input Support
* Voice Output (Text-to-Speech)
* Multilingual Query Detection
* Real-time Chat Interface
* Chat History Management
* Dark and Light Theme Support
* Responsive UI Design
* Machine Learning Insights Dashboard

## Machine Learning Workflow

User Query
→ TF-IDF Vectorization
→ Logistic Regression Intent Classification
→ Intent Detection
→ Prompt Enhancement
→ Groq LLM
→ AI Response Generation
